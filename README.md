# ecom-shop
Для початку роботи з проєктом слід виконати такі кроки:
1) Скопіювати проєкт - ```git clone https://github.com/ost1qq/ecom-shop.git```
2) Встановити всі залежності - ```npm install```
3) Запустити фронтенд - ```npm run dev```
4) Запустити json server - ```npm run server```

To get started with the project, follow these steps:
1) Clone the project - ```git clone https://github.com/ost1qq/ecom-shop.git```
2) Install all dependencies: - ```npm install```
3) Run the frontend - ```npm run dev```
4) Start the JSON server: - ```npm run server```
